# Product Requirements Document
## AI Walkthrough-to-Bid Copilot for Commercial Cleaning Companies

### Product goal
Reduce the time and friction required for a small commercial cleaning company to convert walkthrough inputs into a client-ready draft bid package.

### Core promise
From walkthrough to draft bid package in 10 minutes.

### Target user
Primary user:
- owner
- office manager
- estimator
- sales/admin person at a small commercial cleaning business

Best-fit companies:
- 2–25 cleaners
- custom quoting process
- commercial accounts
- owner-led or lightly staffed back office

### Problem statement
After a site walkthrough, cleaning companies often have scattered voice notes, photos, typed notes, and partial templates. Turning that material into a clean scope and proposal is slow, repetitive, and error-prone.

### Jobs to be done
Functional:
- organize raw walkthrough info
- identify missing info before sending a bid
- draft a scope of work
- generate an area-by-area checklist
- produce a professional proposal draft
- create a follow-up email

Emotional:
- feel organized after walkthroughs
- avoid embarrassing omissions
- sound more professional
- respond faster to prospects

### MVP success criteria
A user can:
1. create a walkthrough record
2. upload or paste raw inputs
3. review extracted structured details
4. generate a usable first-pass bid package
5. copy or export outputs with minimal edits

### Primary metric
Median time from raw walkthrough input to usable draft package.

### Secondary metrics
- number of walkthroughs created
- percent that reach output generation
- repeat usage per company
- average edits before copy/export
- user-reported time saved
- conversion from first to second walkthrough

### Inputs
- client name
- property name
- property type
- square footage
- number of floors
- service frequency
- occupancy notes
- special requirements
- typed notes
- voice memo upload
- photo upload

### AI processing
The system will:
- transcribe audio
- summarize image observations
- extract structured building/service details
- identify missing information
- generate assumptions and clarification questions
- draft output documents

### Outputs
- structured site summary
- scope of work draft
- room/area checklist
- proposal draft
- assumptions/missing info list
- client follow-up email
- internal questions to confirm

### Non-goals for v1
Do not build:
- scheduling
- dispatch
- invoicing
- payroll
- employee mobile app
- inspection workflows
- route optimization
- autonomous pricing engine
- QuickBooks sync

### Screen list
1. Landing page
2. Dashboard
3. New walkthrough
4. Review extracted details
5. Outputs page
6. Company settings

### Build principles
1. Structure before generation
2. Surface uncertainty clearly
3. Vertical polish beats general AI cleverness
